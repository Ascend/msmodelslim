# Multimodal Understanding Model Quantification

Multimodal Understanding Model, also known as Visual Language Model (Vision-Language Models, VLM), has powerful functions. It can process multiple data types such as images, videos, and text, and perform multiple downstream tasks such as understanding image content and generating corresponding natural language descriptions.

## Preparation Before Use

 * Use the matching CANN version 8.2.RC1 or later.
 * Install the msModelSlim tool. For details, see.[msModelSlim Installation Guide](../../docs/zh/getting_started/install_guide.md)    .
 * **Different multi-modal understanding models depend on different transformers versions and third-party libraries. For details, see the quantization description of each multi-modal understanding model.**
