# GLM Quantification Description

## Model Introduction

 * [GLM](https://github.com/THUDM/GLM)    It is an open source version of the latest generation of pre-trained models GLM-4 series launched by Smart Spectrum AI. GLM-4-9B and its human preference-aligned version GLM-4-9B-Chat all outperformed Llama-3-8B in semantic, mathematical, reasoning, code, and knowledge dataset evaluations. In addition to the ability to conduct multiple rounds of conversations, the GLM-4-9B-Chat also features web browsing, code execution, custom tool calls, and long text inference (supporting advanced features such as up to 128K contexts). This generation model adds multi-language support, supporting 26 languages including Japanese, Korean, and German. We also introduced the GLM-4-9B-Chat-1M model supporting 1M context length (about 2 million Chinese characters) and the GLM-4V-9B multimodal model based on GLM-4-9B. The GLM-4V-9B has the capability of multi-round dialogue in both Chinese and English at a high resolution of 1120 x 1120. It has been used in multi-modal evaluation of comprehensive Chinese and English skills, perceptual reasoning, text recognition, and chart understanding. The GLM-4V-9B exhibits superior performance over the GPT-4-turbo-2024-04-09, Gemini 1.0 Pro, Qwen-VL-Max and Claude 3 Opus.

## Use Example

 * Replace \{floating-point weight path\} and \{quantization weight path\} with actual paths.
 * If NPU multi-card quantization is required, configure environment variables to support multi-card quantization. GLM-4-9B quantization requires only a single card.
    
    ```shell
    export ASCEND_RT_VISIBLE_DEVICES=0,1,2,3,4,5,6,7
    export PYTORCH_NPU_ALLOC_CONF=expandable_segments:False
    ```

 * To load a custom model, call`from_pretrained`To be specified when the function`trust_remote_code=True`The modified custom code file can be loaded correctly. (Ensure the security of the loaded custom code file.)

### Quantification of GLM-4-9B Model

#### GLM-4-9B W8A8C8 Quantification

 * Generate the weight of the GLM-4-9B model w8a8c8 in histogram quantization mode and perform the calculation on the NPU.
    
    ```shell
    python3 quant_glm.py --model_path {浮点权重路径} --save_directory {W8A8C8量化权重路径} --device_type npu --act_method 2 --disable_level L0 --w_bit 8 --a_bit 8 --use_kvcache_quant True --calib_file ../common/mix_dataset_glm.json --anti_file ../common/mix_dataset_glm.json --trust_remote_code True
    ```
