# InternLM2 Quantification Case

## Model Introduction

 * InternLM open source several of the InternLM family's basic models and chat models tailored for real-world scenarios. A set of InternLM inference model based on NPU hardware is implemented in this code base. Works with the acceleration library to achieve the ultimate inference performance on the NPU.

## Preparation Before Use

 * Install the msModelSlim tool. For details, see.[msModelSlim Installation Guide](../../docs/zh/getting_started/install_guide.md)    .

## Use Example

 * Replace \{floating-point weight path\} and \{quantization weight path\} with actual paths.
 * To use NPU multi-card quantization, configure environment variables first.
    
    ```shell
    export ASCEND_RT_VISIBLE_DEVICES=0,1,2,3,4,5,6,7
    export PYTORCH_NPU_ALLOC_CONF=expandable_segments:False
    ```

 * To load a custom model, call`from_pretrained`To be specified when the function`trust_remote_code=True`The modified custom code file can be loaded correctly. (Ensure the security of the loaded custom code file.)

### InternLM2-20B

#### InternLM2-20B W8A8 Quantification

```shell
python3 quant_internlm2.py --model_path {浮点权重路径} --save_directory {W8A8量化权重路径} --w_bit 8 --a_bit 8 --device_type npu --trust_remote_code True
```

#### InternLM2-20B W8A16 Quantification

```shell
python3 quant_internlm2.py --model_path {浮点权重路径} --save_directory {W8A16量化权重路径} --w_bit 8 --a_bit 16 --device_type npu --anti_method m1 --trust_remote_code True
```

#### InternLM2-20B KV Cache W8A8 Quantification

```shell
python3 quant_internlm2.py --model_path {浮点权重路径} --save_directory {W8A8C8量化权重路径} --w_bit 8 --a_bit 8 --device_type npu --use_kvcache_quant True --disable_level L5 --trust_remote_code True
```
