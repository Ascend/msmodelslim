# Quantification Description of Hunyuan

## Model Introduction

 * [Tencent-Hunyuan-Large](https://github.com/Tencent/Tencent-Hunyuan-Large)    Currently, it is the largest open-source MoE model based on Transformer in the industry. It has 389 billion parameters and 52 billion active parameters. It has high-quality synthetic data enhanced training, KV cache compression, expert-specific learning rate scaling, and long context processing capabilities. (The pre-trained model supports 256 KB text sequences, and the Instruct model supports 128 KB text sequences.)

## Preparation Before Use

 * Install the msModelSlim tool. For details, see.[msModelSlim Installation Guide](https://gitcode.com/Ascend/msmodelslim/blob/26.0.0/docs/en/getting_started/install_guide.md)    .
 * For the transformers version, the value must be greater than or equal to 4.48. 2.

## Use Example

 * Replace \{floating-point weight path\} and \{quantized weight path\} with actual paths.
 * To use NPU multi-card quantization, configure environment variables first to support multi-card quantization.
    
    ```shell
    export ASCEND_RT_VISIBLE_DEVICES=0,1,2,3,4,5,6,7
    export PYTORCH_NPU_ALLOC_CONF=expandable_segments:False
    ```

 * To load a custom model, call`from_pretrained`To be specified when the function`trust_remote_code=True`The modified custom code file can be loaded correctly. (Ensure the security of the loaded custom code file.)

### Hunyuan-Large

#### Check Before Running

The Hunyuan-Large model is large and requires manual adaptation. To avoid wasting time, modify the related content according to the following mandatory check items before running the script:

 * 1. Update transformers version (> = 4.48. 2) is required.

#### Hunyuan-Large W8A8 Hybrid Quantification (Experts layer: W8A8 Dynamic Quantization; other layers: W8A8 Quantization)

Note: You need to run the command in the current script directory.

 * Generate the W8A8 hybrid weight of the Hunyuan-Large model.
    
    ```shell
    python3 quant_hunyuan.py --model_path {浮点权重路径} --save_directory {量化权重路径} --anti_method m4 --trust_remote_code True
    ```

#### Hunyuan-Large Quantification QA

 * Q: Error if metadata.get("format") not in \["pt", "tf", "flax", "mix"\]: AttributeError: "NoneType" object has no attribute'get' is reported in modeling_utils.py.
 * A: indicates that the metadata field is missing in the input weight. You need to install and update the transformers version (≥ 4.48. 2).
